# Abdullah's Developer Portfolio

This is a personal developer portfolio website built using **HTML, CSS, and JavaScript**. It demonstrates modern web development and design skills with interactive features, responsive layout, and smooth animations.

## Features

- Dark/Light Mode toggle
- Smooth scrolling navigation
- Typed.js animated text
- Skills section with hover effects
- Projects section with neon hover effect
- Contact form with styled inputs and button
- Fully responsive layout (mobile, tablet, desktop)
- Glassmorphism and neon design theme

## Skills Highlighted

- HTML5, CSS3, JavaScript (ES6+)
- Responsive Design, Flexbox, Grid
- UI/UX Design
- Git & GitHub
- Bootstrap / Tailwind CSS
- React.js, Node.js / Express.js
- Database Basics (MySQL/MongoDB)
- Figma / Adobe XD
- Photoshop / Illustrator
- Video Editing
- MS Office Expertise

## How to Use

1. Clone or download this repository.
2. Open `index.html` in a web browser.
3. Use the navigation menu to explore different sections.
4. Toggle between Dark and Light modes using the buttons.
